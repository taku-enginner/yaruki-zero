# YARUKI ZERO - やる気グラフ（全部ゼロ）

![screenshot](./screenshot.png)

## 概要

「やる気」を記録するアプリ…なのに、**グラフにはすべてゼロで表示される**という完全ネタ系モチベーショントラッカーです。  
それでも記録は残り、皮肉なコメントがあなたを優しく煽ってくれます。

---

## 使用技術

- Ruby on Rails 7
- PostgreSQL（Docker構成）
- Tailwind CSS
- Chart.js
- Devise + OmniAuth（Google認証）
- ActionMailer（登録時 or 毎朝レポート送信）
